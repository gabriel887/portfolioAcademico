2º Semestre - Site Social

O site social foi um projeto proposto na disciplina de Padrões de projeto de sítios Internet II durante o semestre de 2021. O objetivo foi o desenvolvimento de uma landpage para uma ONG social da cidade que não possuísse uma ou o desenvolvimento de uma landpage referente a um mês de conscientização, por exemplo: Setembro Amarelo, Outubro Rosa ou Fevereiro Roxo. O principal objetivo era o aprendizado de framework CSS para a criação de sites e o desenvolvimento de idealizações de sites.

O tema escolhido foi o Fevereiro Roxo, o mês de conscientização do Alzheimer, Fibromialgia e Lúpus, para o desenvolvimento do projeto foi utilizado o HTML5, CSS3 e o framework Materialize. Para atrair os leitores, foi utilizado o efeito Parallax, a disposição de uma imagem com um texto em cima, para trazer imagens de alta qualidade que atraiam a atenção do usuário para a conscientização do problema.

3º Semestre - Ecommerce Livraria

O E-commerce foi um projeto avaliativo realizado de maneira individual proposto na disciplina de Programação de sítios para Internet I durante o semestre de 2022. Seu objetivo era o desenvolvimento de uma pequena simulação de E-commerce, requerendo a escolha de um ramo mercadológico para o desenvolvimento de uma página contendo produtos para que fosse gerado um carrinho de compra que pudesse adicionar e remover produtos através da linguagem JavaScript. O principal objetivo da atividade era aprender a pegar valores de elementos HTML e conseguir criar e inserir elementos HTML através do JS, assim como removê-los.

Neste trabalho foram utilizadas as linguagens de HTML, CSS, Javascript e o framework Bootstrap para aplicar a utilização do componente Modal. A estilização foi feita utilizando CSS3 e medias queries para gerar um website responsivo que possa ser utilizado tanto em computadores quanto aparelhos móveis.

4º Semestre - Projeto Lins Turismo

O Projeto Lins Turismo foi uma atividade proposta na disciplina de Projeto de Navegação e Interação durante o segundo semestre de 2022, tendo por objetivo a realização da prototipação de uma landpage para a seção de turismo da cidade de Lins. O projeto teve sua ideia desenvolvida em conjunto com a prefeitura de Lins, por isso havia a necessidade de realizar a coleta de requisitos e estar desenvolvendo de acordo com as requisições do cliente.

Foi utilizado para o trabalho o aplicativo Figma para criação do protótipo do website, com utilização de pesquisas como da teoria das cores para desenvolvimento do layout da página.

5º Semestre - Projeto Muzzik

O projeto Muzzik foi desenvolvido para a disciplina de Tópicos Especiais II durante o semestre de 2023, com o objetivo de oferecer um aplicativo reprodutor de músicas que foram baixadas no celular, para oferecer uma maneira simples de organizar as músicas e categorizá-las.

Para a prototipação e idealização do projeto foi utilizada a metodologia Object Oriented Hypermedia Design Method (OOHDM) para que as etapas de criação sejam propriamente documentadas e sejam facilmente visualizadas.

Já a implementação consta com HTML 5, CSS3 e JavaScript, com a utilização dos frameworks do Cordova, que utiliza o JavaScript para acessar as ferramentas nativas de dispositivos móveis e o Jquery Mobile, que oferece uma maior praticidade no desenvolvimento de aplicativos mobiles.
